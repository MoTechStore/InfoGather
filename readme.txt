This is tool used in the earl phases of information gathering and footprinting in PenTest and most hacking courses.

ls to see the content of the folder InfoGather

Adding Excutable permission to the Script
 sudo chmod +x InfoGather.sh


Installing the requirements
pip install -r requirements.txt



After That run the script InfoGather.sh
./InfoGather.sh
After the script to run you can type in ip address of the target or domain name for the trial type 127.0.0.1
and see the results first.
